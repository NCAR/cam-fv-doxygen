program cam_dart_obs_preprocessor

Simple program that opens an obs_seq file and loops over the observations and copies them to a new output file. This is intended to be a template for programs that want to alter existing obs in some simple way.

NOTE: Observations in a sequence file are processed in ascending time order. For efficiency at creation time, sequence files support observations in a different physical order than temporal order. This program will reorder the observations to be in temporal order, so the output observations may have a different observation index number than in the input file.
